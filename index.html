Accordion Component

An accessible accordion component that follows WCAG guidelines.

This is the content for section 1. The accordion component allows you to show and hide content in a collapsible panel.

This is the content for section 2. When you click on the header, this content will toggle visibility.

This is the content for section 3. The accordion is fully accessible and can be controlled via keyboard.

HTML Structure

<div class="c-accordion">
    <button class="c-accordion__trigger" aria-expanded="false">Accordion Section</button>
    <div class="c-accordion__content">
        <p>This is the accordion content.</p>
    </div>
</div>

JavaScript Initialization

// Initialize all accordions on the page
document.querySelectorAll('.c-accordion').forEach(accordion => {
    new A11yAccordion(accordion);
});

Tree View Component

An accessible tree view navigation component that follows WAI-ARIA best practices.

  • Home
  • Products
    • Category 1
      • Product A
      • Product B
    • Category 2
  • About
    • Our Story
    • Team
  • Contact

HTML Structure

<nav class="tree-view-nav">
    <ul class="menu menu--level-1">
        <li class="menu-item">
            <a href="#" class="menu__link menu__link--level-1" aria-role="tree-item">Parent Item</a>
            <ul class="menu menu--level-2">
                <li class="menu-item">
                    <a href="#" class="menu__link menu__link--level-2" aria-role="tree-item">Child Item</a>
                </li>
            </ul>
        </li>
    </ul>
</nav>

JavaScript Initialization

// Initialize the tree view navigation
document.querySelectorAll('.tree-view-nav').forEach(nav => {
    new A11YTreeViewNavigation(nav);
});
